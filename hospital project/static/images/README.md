# Adding Images to the Homepage

## Patient-First Section Image

To add the image for the "Patient-First & Trust-Driven" section:

1. **Save your image file** in this folder (`static/images/`)
2. **Name it**: `patient-care-image.jpg` (or `.png`, `.jpeg`)
3. **Supported formats**: JPG, PNG, JPEG

### Current Image Path:
The homepage expects the image at:
```
/static/images/patient-care-image.jpg
```

### Alternative Image Names:
If you use a different filename, update the path in:
- `templates/index.html` (line ~40)

### Image Recommendations:
- **Size**: 800x600px or larger
- **Format**: JPG or PNG
- **Content**: Should show healthcare professionals and patients (as described)

---

**Note**: If the image is not found, a beautiful placeholder with medical icons will be displayed instead.
